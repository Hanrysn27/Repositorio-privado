bit:  Bit é uma abreviação de binary digit (dígito binário). É a menor unidade de armazenamento do pc, podendo ter dois valores possiveis 0 ou 1

Shell: é uma interface de usuário para acessar os serviços de um sistema operacional, Cmd ou Windows powershell, ou no MAC.OS e Linux é Terminal

SSH: Secure Shell (casca de proteção) um protocolo usado para entrar na hospedagem de um site e fazer os mais variados tipos de mudanças nas configurações.
